== Description ==
Modular, data-agnostic ESG Framework Engine for WordPress. Standardizes corporate sustainability evaluation across different market paradigms using strict machine English JSON schemas.

The plugin implements a decoupled architecture loading schema profiles on the fly from the internal centralized storage matrix.

== Architecture Matrix ==
* **openesea/**: Global quantitative metrics (Core Standard Engine).
* **sdg/**: International UN Sustainability goal mappings.
* **pgs/**: Participatory Guarantee Systems for trust-networks (Solidarity GAS criteria).
* **products/**: Decoupled horizontal vertical market blueprints (e.g., bakery specifications).

== Installation ==
1. Upload the `wp-esg` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Verify the Framework Auto-Discovery notification status bar on the admin dashboard.
